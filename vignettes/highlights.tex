%% To get pretty code highlights without using explicitely 
%% Rmd proceed as followed:
%%   1. Create an ordinary Rmd file 
%%   2. Set the parameter 'keep_tex: TRUE' in the YAML 
%%      header
%%   3. Open the created .tex file and copy and paste
%%      everything in the Shaded environment, This stuff
%%      was created from the code Chunk. 
%%   4. A bit more advanced is an automated size changer
%%      with a hook function in markdown (not that hard)
